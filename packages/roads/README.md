# roads

> Roads is the commit queue service, which handles merging pull requests inside
> the big repositories.

## Usage

### Quickstart

```sh
docker run --publish 5000:5000 --rm --env GH_TOKEN=${GH_TOKEN} ghcr.io/siberianmh/roads:latest
```
