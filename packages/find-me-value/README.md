# find-me-value

## Install

```sh
$ npm install find-me-value
or
$ yarn add find-me-value
```

## Usage

```ts
import { findMeValue } from 'find-me-value'

findMeValue('Module Version: v1.0.0', 'Module Version')
//=> v1.0.0

findMeValue('Foo-Bar: Anything', 'Foo-Bar')
//=> Anything
```
